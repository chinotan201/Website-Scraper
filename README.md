# ‚≠ê **DESCRIPTION**

**Website Scraper** is a tool designed to automate the process of collecting table data from websites. It allows you to select specific columns, extract structured data, and download it as CSV files. The scraper can also periodically check for updates, making it easy to track changes in table content over time.

This tool is ideal for:
- **Data collection for research or reporting**
- **Automating repetitive download tasks**
- **Monitoring table content changes on websites**
